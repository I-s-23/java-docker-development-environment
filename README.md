# Java-docker-development-environment

## OverView

Java Continer with Remote Containers

## Usage

1. VsCodeで「Ctrl + Shift + P」を押下
2. 「Remote-Containers: Open Folder in Container」と入力
3. このリポジトリをCloneしたディレクトリを選択
4. Dockerが起動して、Javaが使用可能になる

## Version

- Java
  - AdaptOpenJDK11
